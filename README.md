🚀 Amazon Cloud Sales Analytics

📌 Project Overview

This project demonstrates an end-to-end cloud analytics workflow using Google BigQuery and Looker Studio to analyze Amazon sales data and generate actionable business insights.

The goal was to transform raw transactional data into a decision-making dashboard for revenue performance and sales trends.

⸻

🛠 Tech Stack

* Google BigQuery (Cloud Data Warehouse)
* SQL (Data Transformation & Aggregation)
* Looker Studio (Dashboard & Visualization)

⸻

📊 Key Metrics

* Total Revenue: ₹78.59M
* Total Orders: 128,975
* Monthly Revenue Trends
* Revenue by Category
* B2B vs B2C Performance

⸻

📈 Key Insights

* Top product categories contributed ~65–70% of total revenue
* Identified clear revenue spikes indicating seasonal demand patterns
* B2B orders showed distinct purchasing behavior compared to B2C
* Certain time periods generated significantly higher order volumes, indicating demand peaks

⸻

🧠 What This Project Demonstrates

* Writing production-level SQL queries in BigQuery
* Designing scalable summary tables and reporting views
* Translating raw data into business insights
* Building interactive dashboards for stakeholders
